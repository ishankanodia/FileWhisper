Get going in a minute

Three steps. Then never the terminal again.

1

Open your terminal

On a Mac, press Command + Space, type “Terminal”, and hit Enter.

Terminal
2

Paste one line

Paste the install command from above and press Enter. It sets everything up and adds a FileWhisper icon to your Desktop.

3

Open & ask

Double-click FileWhisper. It opens in your browser — pick a folder, then just type your question in plain English.

Faster & more accurate

Paste a key from any provider — Groq, OpenAI, Claude, Gemini, or any OpenAI-compatible API — in the LLM Settings panel for noticeably quicker, sharper answers. Totally optional.

Why FileWhisper

Private by design, simple by default.

100% local

Indexing, search and OCR all happen on your machine. Files never leave your computer.

No setup headaches

One command. No Docker, no databases, no Python wrangling. Small install, no PyTorch.

Reads almost anything

PDFs, text and Markdown, plus scanned pages and images via built-in OCR.

Always cites sources

Every answer shows exactly which of your files it came from, so you can trust it.
